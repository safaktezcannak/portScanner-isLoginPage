With this Python script, you can scan the ports of a single IP or IP ranges that you can export via terminal or txt file with the nmap module.
It will show you the http and https ports, and then request the source codes of these ports with the request module.
It will look for the password string in the source code and check whether this page is the login page. 
It will give you all this data as an output.txt file.
When you give it as an ip range, you can ping scan these ips.
